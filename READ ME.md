Bouquet Maker (v2.0)

A web-based Flower Recommendation & Bouquet Generator that suggests specific flowers based on the user's emotional intent and color preference. This project was developed as a final project for coursework at Modesto Junior College.

Features
  - Flower Generator: Match a feeling (e.g., "love", "happy", "strength") with a color to discover the perfect flower.
  - Flower Dictionary: An educational categorical view of flowers grouped by emotional themes like "Grief," "Love," and "Just Because".
  - Interactive Map: Integration with the Google Maps API to help users find local flower shops in Modesto, CA.
  - Pixel Art Aesthetic: Custom-styled UI featuring pixel-art rendering for flower imagery.
  - Responsive Tabs: A single-page application (SPA) feel using CSS and Vanilla JavaScript tab logic.

Tech Stack
  - Frontend: HTML5, CSS3
  - Scripting: Vanilla JavaScript (ES6+)
  - APIs: Google Maps JavaScript API (including Places and Markers)
  - Assets: Custom pixel art images for various floral species
  
Project Structure
  - index.html      # Main structure and UI components
  - script.js       # Logic for flower matching, tab switching, and Map API
  - style.css       # Custom styling and layout (flexbox/fixed positioning)
  - images/         # Directory containing floral pixel art
  - default.jpg     # Placeholder image for null results

How to Run Locally
  1. Clone the repository: git clone https://github.com/your-username/bouquet-maker.git
  2. Open the project: Open index.html in any modern web browser.
  3. API Note: The Google Maps integration requires a valid API key. Ensure the key in the index.html script tag is active or replace it with your own credentials.

Usage
  - Navigate to the Flower Generator tab.
  - Select a color from the dropdown menu.
  - Type a feeling (e.g., innocence, mystery, or melancholy) into the text box.
  - Click Submit to see the suggested flower and its image appear.
